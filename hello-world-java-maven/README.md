# Hello world with java and maven

[![MIT license](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE.md)

Example app with default options but backend: java, management: maven.

This project was generated with [swaaplate](https://github.com/inpercima/swaaplate) version 2.5.1.

## Prerequisites

### Angular CLI

* `angular-cli 15.0.4` or higher

### Java

* `jdk 17` or higher

### Node, npm or yarn

* `node 18.10.0` or higher in combination with
  * `npm 8.19.2` or higher or
  * `yarn 1.22.19` or higher, used in this repository

## Getting started

```bash
# clone project
git clone https://github.com/inpercima/swaaplate-hw
cd swaaplate-hw/hello-world-java-maven
```

### Modules
